# Blasius Laminar Flat Plate — Reference Data

**Case:** `blasius_flat_plate` — zero-pressure-gradient *laminar* flat plate.
**Conditions:** Re_L = 1e5 (based on plate length L = 2.0), M = 0.1, laminar
(no turbulence model). The whole plate is below transition (Re_x < 5e5).
**Tier:** forward-regime credibility (the flapping mission's low-Re regime) —
see `.claude/rules/flapping-validation-ladder.md`.

## `cf.csv` — local skin-friction coefficient

Columns: `x`, `cf`. The reference is the **exact Blasius similarity solution**:

```
Cf(x) = 0.664 / sqrt(Re_x),   Re_x = U·x/ν = Re_L · x / L
```

(H. Blasius, *Grenzschichten in Flüssigkeiten mit kleiner Reibung*, Z. Math.
Phys. 56 (1908); H. Schlichting & K. Gersten, *Boundary-Layer Theory*, 8th ed.,
the laminar flat-plate skin-friction law.) This is **analytical**, not
digitized experimental data — the constant 0.664 is exact, so the reference
carries **no digitization/input uncertainty** (`u95_input = 0`). The CSV
tabulates the law on a grid strictly inside the developed-plate window
`[0.1, L]`, regenerated by `aero.vv.forward_regime.blasius_flat_plate.blasius_cf`.

The harness samples the solved Cf along the plate (the `surfaces` wall sampler →
`wall_distribution`), drops the leading-edge window (Cf ~ x^-1/2 is singular at
x=0), and compares pointwise against this law (5% relative tolerance, a
contract — a failure is a laminar-discretisation regression, never relaxed).

## Not yet covered (follow-up)
